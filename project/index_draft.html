CS Dealership. Provo, Utah - Project Site Plan

Site Name

Name: CS Dealership. Provo, Utah

Reason: The name reflects the dealership’s focus on trucks and SUVs, specializing in models like the F150, F250, and 4x4 SUVs. It's short, memorable, and suitable for branding purposes.

Domain: cs-dealership.com

Site Purpose

The purpose of this website is to provide customers with access to information on available trucks and SUVs, financing options, and dealership location. It’s designed to help customers in Utah find the right vehicle for their needs and contact CS Dealership. Provo, Utah easily.

Scenarios

  • What 4x4 models are best for off-road conditions in Utah?
  • How can I get in touch with CS Dealership. Provo, Utah for a test drive?

Color Schema

Primary Color: Dark blue (#081534) is used for the header and navigation, giving a professional look.

Secondary Color: Medium blue (#133a7c) as an accent, adding subtle contrast where needed.

Text Color: Black (#000000) for the main text, ensuring readability against the lighter background.

Background Color: White (#ffffff) for the body, providing a clean, open layout.

Typography

Primary Font: Montserrat for headings to give a bold, modern look.

Secondary Font: Roboto for body text to ensure readability and a clean appearance.

Wireframe

Mobile View: Collapsible menu, featured trucks in a scrollable list, and prominent buttons for “Inventory” and “Contact Us.”

Desktop View: Full-width hero image, top menu, grid display for featured vehicles, and a contact form at the bottom.
