CS Dealership. Provo, Utah - Project Site Plan

Site Name

Name: CS Dealership, Provo, Utah (CS stands for Canyon Street, reflecting the location)

Reason: The name reflects the dealership’s location, street, county, and state to appropriately target customers.

Domain: cs-dealership.com

Site Purpose

The purpose of this website is to provide customers with access to information on available trucks and SUVs, financing options, and dealership location. It’s designed to help customers in Utah find the right vehicle for their needs and contact CS Dealership. Provo, Utah easily.

Scenarios

  • What 4x4 models are best for off-road conditions in Utah?
  • How can I get in touch with CS Dealership. Provo, Utah for a test drive?

Color Schema

Primary Color: Dark blue (#081534) is used for the header and navigation, giving a professional look. THis is color is used in big screen

Secondary Color: Medium blue (#133a7c) is also used for the header and navigation, but in small screens

Text Color: Black (#000000) for the main text, ensuring readability against the lighter background.

Background Color: White (#ffffff) for the body, providing a clean layout.

Typography

Primary Font: Montserrat for headings and footer to give a bold, modern look.

Secondary Font: Ubuntu Mono for body text to ensure readability and a clean appearance.

Wireframe

Mobile View: Collapsible menu, featured trucks in a scrollable list, and prominent buttons for “Inventory”, “Contact us”, "About us" and "VIN tool" hidden in a hamburger button

Desktop View: Top menu, grid display for information and news, inventory with a grid of vehicles in stock with a modal providing information, contact us page with a form, contact information, and a form that uses the GET method, VIN tool page that uses an API to display useful information about any car in the market
